Tripscape

Designing a landing page for a travel itinerary and guide platform

Tripscape is a platform that lets people create and store travel itineraries, write and discover travel guides, search for inspiration, and track how much of the world they've explored.

This page documents the process of designing Tripscape's landing page across 25 iterations — each exploring a different combination of layout, mood, visual style, audience, and tone — and explains how I arrived at a final design.

Chapter 1

Chapter 2

Shifting to a Logged-In Experience

Iteration 01 was a marketing page for new users. But what does Tripscape actually feel like once you're using it? I shifted to designing the logged-in dashboard — the page travelers see every day.

Chapter 4

Exploring Travel Themes

With the core structure established, I spent several iterations exploring different travel-themed aesthetics — from leather notebooks to retro airline posters to scrapbooks and boarding passes. I experimented with combining themes, adding new features like search bars and trip countdowns, and refining what worked. The layouts stayed similar across these iterations, but the visual identity evolved significantly.

Chapter 5

Breaking Out of the Format

I realized the previous iterations were all variations of the same underlying layout — just with different color palettes and card styles swapped in. The format had become a rut. I challenged myself to start from scratch with completely new structures: newspaper columns, manga panels, Japanese architectural spaces. This chapter is about breaking the template and finding genuinely different ways to organize the same information.

09

Terracotta Polaroid

Wide single column Warm & clean Mediterranean modern All travelers Inviting

The last iteration before the format break. Wider 1200px layout, terracotta + slate palette, combined Discover section with filter pills. Cleaner, but still the same underlying structure — this is when I realized I needed to start over.
